# sdmux driver

`jumpstarter-driver-sdmux` provides functionality for using the SDmux storage
multiplexer. This device multiplexes an SD card between the DUT and the exporter
host.

## Installation

```{code-block} console
:substitutions:
$ pip3 install --extra-index-url {{index_url}} jumpstarter-driver-sdmux
```
